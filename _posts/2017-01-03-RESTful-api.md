RESTful是一种互联网软件框架，2000年由Roy Thomas Fielding提出。
    
<!-- more -->

一、定义
REST是Representational State Transfer的简称，中文翻译是“表现层状态转化”。

二、设计RESTful api注意点

1. 将api部署在专用的域名之下

2. 将版本信息放入URL

3. 对资源的操作，由HTTP动词表示

4. 使用Query过滤数据

5. API的身份认证应该使用OAuth 2.0框架
